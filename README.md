☕ Coffee Shop Sales Analytics Dashboard (Excel)

An interactive business intelligence dashboard built entirely in Microsoft Excel to analyze sales performance, customer behavior, and product trends across multiple coffee shop locations.

📖 Overview

The Coffee Shop Sales Analytics Dashboard is an end-to-end Excel-based Business Intelligence project that transforms raw sales data into interactive visual insights. Using Pivot Tables, Pivot Charts, Slicers, and KPI cards, the dashboard enables users to monitor sales performance, customer traffic, product demand, and purchasing behavior across different locations and time periods.

The dashboard provides business owners with actionable insights to optimize inventory, improve staffing decisions, and identify top-performing products.
